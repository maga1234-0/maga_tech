# **App Name**: MagaTech Portfolio

## Core Features:

- Hero Section: Display the 'MagaTech' brand name with a tech-focused tagline and CTA buttons (View Projects, GitHub, Contact).
- Projects Section: Showcase projects in a grid-based layout with title, description, tech stack, and GitHub link.
- GitHub Integration: Provide a link to the GitHub profile, with an optional feature to fetch and display pinned repositories using the GitHub API.
- About Section: Present a short professional bio, focusing on technology, innovation, and development.
- Contact Section: Include a WhatsApp button linking directly to the specified WhatsApp number and an email button using mailto.
- Portfolio Optimizer: Use generative AI tool to analyze project descriptions and tech stacks to suggest improvements for better presentation and SEO.

## Style Guidelines:

- Primary color: Deep blue (#3872A8), aligning with a tech-focused theme and evoking trust and reliability.
- Background color: Dark gray (#28292B), providing a modern, dark tech UI.
- Accent color: Emerald green (#50C878) to highlight interactive elements and CTAs, providing a vibrant contrast against the dark background.
- Headline font: 'Space Grotesk', sans-serif, for headlines and short amounts of body text. Body font: 'Inter' for longer text.
- Utilize Lucide Icons for a consistent and modern look across the portfolio.
- Implement a mobile-first, fully responsive layout, ensuring fast loading times and accessibility.
- Use Framer Motion for subtle hover animations on project cards and smooth transitions between sections.
