# MagaTech Portfolio

This is a modern, responsive, and feature-rich portfolio website for MagaTech, built with Next.js and Firebase Studio. It showcases projects, skills, and provides an easy way for visitors to get in touch.

## Features

- **Modern & Responsive Design**: A sleek, dark-themed UI that looks great on all devices, from desktops to mobile phones.
- **Project Showcase**: A dedicated section to display your work, with project details, tech stacks, and links to GitHub.
- **AI-Powered Content Optimization**: An integrated AI feature to help you improve your project descriptions and SEO keywords on the fly.
- **Dynamic Skills Section**: An animated display of your technical skills.
- **Contact Section**: Simple and direct contact options via WhatsApp and Email.
- **Test on Phone**: A built-in QR code generator to easily preview the live site on a mobile device.

## Technology Stack

- **Framework**: [Next.js](https://nextjs.org/) (with App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) with [ShadCN UI](https://ui.shadcn.com/) components
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **AI Integration**: [Genkit](https://firebase.google.com/docs/genkit) for generative AI features
- **Deployment**: Ready for [Firebase App Hosting](https://firebase.google.com/docs/app-hosting)

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- Node.js (v18 or later)
- npm or yarn

### Installation

1.  **Clone the repository:**
    ```sh
    git clone https://github.com/maga1234-0/maga_tech.git
    cd maga_tech
    ```

2.  **Install NPM packages:**
    ```sh
    npm install
    ```

3.  **Run the development server:**
    ```sh
    npm run dev
    ```

Open [http://localhost:9002](http://localhost:9002) with your browser to see the result. You can start editing the page by modifying `src/app/page.tsx`.
